# 🧵 Вышивальная студия — сайт-визитка

Учебный проект: одностраничный сайт вышивальной студии в современном стиле.

🔗 **Живая версия:** https://a750ak-blip.github.io/vyshivaem-studio/

## Что использовано
- **HTML** — семантическая структура страницы
- **CSS** — Flexbox, CSS Grid, адаптивная вёрстка (@media)
- **Glassmorphism** — эффект стекла (backdrop-filter), градиенты, свечение, hover-анимации
- **JavaScript** — обработка формы заявки, проверка полей, события

## Чему научился на проекте
- Вёрстка с нуля и адаптив под мобильные
- Современный визуальный стиль (тёмная тема, стекло, неоновые акценты)
- Базовый JavaScript: события, условия, работа с данными формы
